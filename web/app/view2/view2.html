How to test

Rest API'et kan testes via Postman:


Create User

url: {root-url}/api/createuser

Input: JsonObject med 'username', 'password' og 'roles', ex:

{ "username":"user", "password":"test", "roles":["User"] }

Get Users (Admin only)

url: {root-url}/api/admin/users

Output: JsonObject med liste af users, ex:

[ { "username":"user", "roles":[ "User" ] }, { "username":"admin", "roles":[ "Admin" ] }, { "username":"user_admin" "roles":[ "Admin", "User" ] } ]

Delete User (Admin only)

url: {root-url}/api/admin/user/{id}

Input: PathParam med id for den bruger der skal slettes
